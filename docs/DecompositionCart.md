# Декомпозиция эпика


## Модуль 1:

#### Верстка главного экрана корзина

- Пустой экран с плейсхолдером "Корзина пуста"
- Создание списка для отображения NFT
- Верстка ячейки NFT (Изображение, название, рейтинг, цена, кнопка удаления)
- Верстка модального отображения NFT, кнопка удалить/вернуться
- Верстка нижней панели (Кол. NFT, общей суммой, кнопка к оплате)
- Кнопка для сортировки списка (По цене/По рейтинге/По названию)

#### Логика

- Получение списка NFT
- Удаление из списка NFT (Подтеврждение, обновление списка)
- Обновление общего количества NFT и общей суммы при удалении/добавлении
- Сортировка списка NFT (По цене/По рейтинге/По названию)


## Модуль 2:

#### Верстка экрана выбор способа оплаты 

- Создание таблицы для отображения валют
- Верстка ячейки валюты (картинка, полное название, сокращенное название)
- Верстка нижней панели с сылкой на "Пользовательское соглашение" + кнопка "Оплатить"
- Верстка экрана Успешной оплаты, кнопка возврата в корзину "Вернуться в корзину"

#### Логика

- Выбор валюты, выделение валюты
- Работа с таблицей валют

#### Обработка

- Запрос на оплату и обработка ответа
- Отображение успешной оплаты/Показ алерта "Не удалось произвести оплату" с кнопками "Повторить/Отмена"

#### Навигация

- Переход с корзины на выбор валюты и с выбора валюты в корзину, возвращение с экрана "Успешной оплаты"


## Module 3:

#### Логика работы с сетью

- Получение данных об NFT в корзину
- Отправление об удалении NFT из корзины
- Получение списка валют
- Оплата заказа
